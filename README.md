A modern, high-fidelity professional portfolio engineered for high-performance visual storytelling. This project showcases technical expertise in **Android (Native)**, **Flutter**, and **Kotlin Multiplatform (KMP)** through a production-grade interactive interface.

## 🚀 Key Features

### 1. Architectural System Map
An interactive, node-based visualization of technical competencies. Moves away from static skill lists toward a production-grade system diagram featuring:
- **Marching Spark Animations**: Real-time data flow visuals between architectural layers.
- **Node Intelligence**: Deep-dive panels for Presentation, Business Logic, State, and Infrastructure.
- **Dynamic Layering**: HUD-style overlays with scanning effects and stability readouts.

### 2. Narrative Editorial Bio
A typography-first **About** section that treats personal history as a technical manifesto.
- **No-Box Design**: Uses negative space and hairline blueprint dividers for a clean, professional look.
- **Tactical Metrics**: Quantified impact (98% stability, logic sharing) presented with high-contrast precision.

### 3. Executive Project Showcases
A dual-mode project gallery optimized for deep technical analysis.
- **Artifact View**: Detailed breakdowns of architecture, state management, and core innovations.
- **Responsive Staging**: Smooth transitions between vertical mobile flows and horizontal desktop side-cars.

### 4. Premium Smooth Navigation
- **Momentum Scrolling**: Integrated with **Lenis** for a "buttery smooth" agency-level scroll feel.
- **Interactive Cursor**: Custom magnetic-aware cursor system for enhanced engagement.
- **Theme-Aware Logic**: Intelligent adaptive UI that balances contrast across light and dark modes.

## 🛠 Tech Stack

- **Core**: [Next.js](https://nextjs.org/) (App Router, Turbopack)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Motion**: [Framer Motion](https://www.framer.com/motion/)
- **Scrolling**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)

## ⌨️ Development

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
